Sandbox Query Sandbox

Use these query structures to execute performance tests against the 10,000 records index system:

1. Server-side search & filtering:
GET /odata/v2/northwind/Products?$filter=substringof('Chef', name) and unitPrice gt 20
2. Combined Expand, Filter, Order & Paging:
GET /odata/v2/northwind/Orders?$top=10&$skip=20&$filter=shipCountry eq 'Germany'&$expand=customer,items&$orderby=orderDate desc
3. Raw Category Selection:
GET /odata/v2/northwind/Categories?$filter=substringof('Organic', name)&$inlinecount=allpages
